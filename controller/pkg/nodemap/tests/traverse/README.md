## 使用指南

### 运行测试

```bash
# 运行所有遍历测试
go test ./pkg/nodemap/tests/traverse/...
```

### 添加新测试

1. 使用 `fixtures` 包创建测试网络拓扑
2. 使用 `helpers` 包中的辅助函数
3. 遵循现有的测试模式

## 注意事项

- 遍历测试需要完整的网络拓扑配置
- 测试应该覆盖各种路由场景
- 注意测试性能和资源使用
